- A execução dos testes pode ser realizada clicando sobre a pasta da collection ReqresAPI, posteriormente clicar no botão Run.

Configurações
- Baixar o postman https://dl.pstmn.io/download/latest/win64.
- Instalar o postman.
- Abrir o postman importar a coleção ReqresAPI.postman_collection.json.
- Importar as variáveis goblais workspace.postman_globals.json.
